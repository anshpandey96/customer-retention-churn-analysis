# 📊 Customer Retention & Churn Analysis

## 📌 Overview
This project focuses on analyzing subscription-based customer data to identify **churn patterns**, **retention drivers**, and **customer lifetime trends**.  
The objective is to deliver actionable insights and recommendations that help reduce customer loss and improve retention strategies.

---

## 🛠️ Tools & Technologies
- **Python** → Data cleaning, cohort analysis, churn prediction models  
- **Excel** → Exploratory analysis & pivot tables  
- **Power BI** → Interactive dashboards & visualization  

---

## 🎯 Skills Gained
- Customer Retention & Cohort Analysis  
- Customer Lifetime Value (CLV) Metrics  
- Insight-driven Decision Making  
- Data Visualization & Reporting
